 # AWS Cloud Portfolio Website

## Project Overview

This project demonstrates how to host a secure static portfolio website on AWS using Amazon S3, Amazon CloudFront, Amazon Route 53, and AWS Certificate Manager (ACM).

The website is accessible through a custom domain with HTTPS and follows AWS best practices using Origin Access Control (OAC).

## Deployment Steps

1. Purchased custom domain
2. Created Route 53 Hosted Zone
3. Updated GoDaddy Nameservers
4. Created Amazon S3 bucket
5. Uploaded website files
6. Configured CloudFront Distribution
7. Enabled Origin Access Control
8. Requested ACM SSL Certificate
9. Validated certificate using DNS
10. Created Route53 Alias Record
11. Website available over HTTPS
